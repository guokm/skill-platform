# Frontend Design System

## 适用场景

- 设计团队和前端团队需要沉淀统一组件规范
- 要把页面风格从零散样式升级为可复用设计语言
- 需要为 React 后台、官网或营销页搭建统一骨架

## 使用方式

1. 输入产品目标、核心页面和品牌方向
2. 生成颜色、字体、栅格和组件约束
3. 输出可复用的组件建议与页面结构

## 输出内容

- 组件清单
- 交互与动效建议
- 页面结构和信息层级
