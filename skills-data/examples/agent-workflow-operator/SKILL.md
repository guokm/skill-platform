# Agent Workflow Operator

## 适用场景

- 需要拆分复杂任务为多个子步骤
- 希望把固定流程交给自动化任务反复执行
- 需要同时关注状态、产出和失败重试

## 推荐输入

- 目标结果
- 参与角色
- 每一步输入输出
- 风险与兜底逻辑

## 推荐输出

- 工作流阶段图
- 每个 Agent 的责任边界
- 可执行的自动化顺序
